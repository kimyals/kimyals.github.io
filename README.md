# The Spirit of Kimyal

Landing page **The Spirit of Kimyal** — sebuah situs profil yang memperkenalkan **Suku Kimyal**, salah satu suku asli di Papua, tepatnya di daerah **Yahukimo, Papua Pegunungan**. Situs ini menampilkan profil suku, wilayah administratif, Bahasa Kimyal, serta mempromosikan beragam potensi wisata alam dan budaya.

## ✨ Fitur Utama

- **Desain Responsif** — Tampilan optimal di desktop, tablet, dan perangkat mobile menggunakan Tailwind CSS.
- **Navigasi Interaktif** — Navbar *sticky* dengan efek *scroll*, menu mobile, dan indikator menu aktif otomatis (IntersectionObserver).
- **Hero Section dengan Parallax** — Efek parallax pada latar hero (khusus desktop) untuk tampilan yang dinamis.
- **Smooth Scrolling** — Perpindahan antar-bagian yang mulus melalui menu navigasi.
- **Section Lengkap** — Profil, Wilayah (4 distrik & 7 klasis GIDI), Bahasa Kimyal, Potensi Wisata, Galeri, dan Kontak.
- **Galeri Interaktif** — Carousel gambar dengan modal *lightbox* untuk melihat dokumentasi lebih dekat.
- **Widget WhatsApp** — Formulir chat mengambang yang langsung mengirim pesan ke WhatsApp.
- **Peta Lokasi** — Google Maps *embed* menampilkan wilayah Suku Kimyal di Yahukimo.
- **Tombol Back to Top** — Muncul otomatis saat menggulir ke bawah.
- **Aksesibilitas** — Dukungan `prefers-reduced-motion` dan label ARIA pada elemen interaktif.

## 📍 Tentang Suku Kimyal

- **Lokasi:** Kabupaten Yahukimo, Papua Pegunungan.
- **4 Distrik Administratif:** Sela, Korupun, Duram, dan Kwelemdua.
- **7 Klasis GIDI** (Gereja Injili Di Indonesia) yang mencakup jemaat Suku Kimyal.
- **Bahasa Lokal:** Bahasa Kimyal, dituturkan di Kampung Korupun, Dagi, Duram, Kwan Dua, dan Debula (Distrik Korupun).
- **Akar Yahukimo:** Suku Kimyal bersama Suku Yali, Hubla, dan Momuna menjadi akar penamaan Kabupaten Yahukimo.
- **Potensi Wisata:** Trekking & Hiking, Kerajinan Kapak Batu, Air Terjun, Camping, Arung Jeram, Para Layang, dan potensi penanaman padi.

## 🛠️ Teknologi

- **HTML5** — Struktur halaman *single-page*.
- **Tailwind CSS** (via CDN) — Styling utilitas.
- **Alpine.js** — Interaktivitas ringan (navbar, menu mobile, galeri).
- **Font Awesome** — Ikon.
- **Google Fonts** — Plus Jakarta Sans & Playfair Display.

## 🚀 Cara Menjalankan

Proyek ini adalah situs statis tanpa proses build. Cukup buka `index.html` di browser, atau jalankan server lokal:

```bash
# Menggunakan Python
python -m http.server 8000

# Lalu buka http://localhost:8000
```
